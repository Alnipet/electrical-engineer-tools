# Electrical Engineer Tools

Набор инструментов для электротехнических расчетов, разработанный для инженеров-электриков.

## 🧮 Калькуляторы

### Коэффициент спроса
Расчет коэффициента спроса для рабочего освещения согласно СП 256.1325800.2016.

---

## DEV

### 📝 Начало работы

1. Клонируйте репозиторий
2. Установите зависимости: `npm install`
3. Запустите dev-сервер: `npm run dev`

### 🌐 Деплой

Приложение автоматически деплоится на GitHub Pages при выполнении:
```bash
npm run deploy
```
