# Electrical Engineer Tools

Набор инструментов для электротехнических расчетов, разработанный для инженеров-электриков.

## 🚀 Технологии

- **React 18** - библиотека для создания UI
- **TypeScript** - типизированный JavaScript
- **Vite** - современный сборщик
- **Ant Design** - UI компоненты
- **Chart.js** - визуализация данных
- **SASS** - препроцессор стилей

## 📦 Доступные скрипты

### `npm run dev`
Запуск приложения в режиме разработки с hot-reload.
Откройте [http://localhost:3000](http://localhost:3000) для просмотра.

### `npm run build`
Сборка продакшн версии приложения в папку `build`.
Оптимизировано и минифицировано для лучшей производительности.

### `npm run preview`
Просмотр собранной версии локально.

### `npm test`
Запуск тестов в режиме наблюдения.

### `npm run deploy`
Деплой на GitHub Pages.

## 🧮 Калькуляторы

### Коэффициент спроса
Расчет коэффициента спроса для рабочего освещения согласно СП 256.1325800.2016.

## 🏗️ Структура проекта

```
src/
├── assets/              # Статические ресурсы
├── components/          # Переиспользуемые компоненты
│   ├── Header/
│   ├── Layout/
│   └── Logo/
├── DemandFactorCalculation/  # Функциональность расчета
├── App.tsx              # Корневой компонент
└── index.tsx            # Точка входа
```

## 📝 Начало работы

1. Клонируйте репозиторий
2. Установите зависимости: `npm install`
3. Запустите dev-сервер: `npm run dev`

## 🌐 Деплой

Приложение автоматически деплоится на GitHub Pages при выполнении:
```bash
npm run deploy
```
